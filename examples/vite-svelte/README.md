# Vite + Svelte Example

This is an example application demonstrating the usage of Fedimint client in a Svelte application.

## Deploy preview

You can preview this example live with [StackBlitz](https://stackblitz.com/github/fedimint/fedimint-sdk/tree/main/examples/vite-svelte)

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/fedimint/fedimint-sdk/tree/main/examples/vite-svelte)

## Technologies Used

- @fedimint/core: The Fedimint client library for JavaScript applications
- Vite: A fast build tool and development server. Check the [`vite.config.ts`](./vite.config.ts) for configurations required for handling wasm.
- Svelte

## Prerequisites

- Node.js
- pnpm (recommended) or npm

## Running the Application Locally

```bash
# from the root of the repo

# install dependencies
pnpm i

# run the dev server
pnpm run dev:svelte
```
