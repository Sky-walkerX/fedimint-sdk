## Running the Example Locally

Clone the repo

```sh
git clone -b addSvelte https://github.com/Sky-walkerX/fedimint-sdk.git
cd fedimint-sdk
```

Install the package manager with [Corepack](https://nodejs.org/api/corepack.html) & install dependencies.

```sh
corepack enable
pnpm i
```

Run the Example

```sh
pnpm run dev:svelte
```
